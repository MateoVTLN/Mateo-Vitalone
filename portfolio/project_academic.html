Project Title

Year: 2025
Institution: Politecnico di Milano – MSc Music and Acoustic Engineering
Keywords: signal processing, acoustics, machine learning

Abstract

This project investigates the use of spectral representations and machine learning techniques for automatic music genre classification. The study compares classical machine learning approaches with deep convolutional neural networks trained on mel-spectrograms.

Context and Motivation

Music information retrieval (MIR) relies heavily on robust feature representations. Classical audio descriptors such as MFCCs, chroma, and spectral contrast have been widely used in machine learning pipelines. More recently, convolutional neural networks operating on time–frequency representations have demonstrated improved performance.

Methods

The dataset used is the FMA-small dataset containing 8 music genres. Two methodological approaches were implemented:

  • Classical ML models (Random Forest, SVM)
  • Neural networks trained on mel-spectrograms
  • Feature extraction using the librosa library

Results

Confusion matrix of the CNN classifier

The CNN architecture achieved a validation accuracy of 72%, outperforming traditional machine learning approaches trained on handcrafted features.
